# 论文结构模板库

## 中文硕士论文结构模板

### 总体结构

```
第1章 绪论 (约3000-5000字)
├── 1.1 研究背景与意义 (1500-2000字)
│   ├── 1.1.1 研究背景
│   └── 1.1.2 研究意义
├── 1.2 国内外研究现状 (3000-5000字)
│   ├── 1.2.1 国外研究现状
│   ├── 1.2.2 国内研究现状
│   └── 1.2.3 现有研究不足
├── 1.3 研究内容与目标 (500-800字)
├── 1.4 技术路线 (图+200字说明)
└── 1.5 创新点 (300-500字)
```

### 各章结构要求

#### 第1章 绪论
**写作模式：** 从宏观到微观，从背景到问题

**学术逻辑链：**
```
大背景（全球/国家层面）
  → 具体领域问题
    → 现有研究做了什么
      → 还存在什么不足（研究空白）
        → 本研究要解决什么问题
          → 怎么解决（技术路线）
            → 预期成果（创新点）
```

**可复用模板（研究背景）：**
```
[领域]是[宏观背景]的重要组成部分。近年来，随着[驱动因素]，
[问题]日益突出，已成为制约[发展]的关键因素之一。
[具体数据/事实]表明，[量化说明问题严重性]。
因此，开展[本研究方向]研究具有重要的理论意义和实践价值。
```

**本课题示例：**
```
城市污水管网系统是城市基础设施的重要组成部分。
近年来，随着城镇化进程加快和污水收集范围扩大，
管网系统中碳污染物的赋存与迁移问题日益突出，
已成为制约城市碳减排和碳中和目标实现的关键因素之一。
研究表明，污水管网中碳污染物以固、液、气三种相态存在，
其赋存特征和转化规律直接影响碳核算的准确性和减排策略的制定。
```

**常见错误：**
- 研究背景过于宏大，与具体课题脱节
- 文献综述是简单罗列而非批判性分析
- "研究空白"表述过于主观（"很少有人研究"）
- 技术路线图信息量不足或过于复杂

#### 第2章 材料与方法 (约5000-8000字)
```
第2章 材料与方法
├── 2.1 研究区域概况 (1000-1500字)
│   ├── 2.1.1 地理位置
│   ├── 2.1.2 气候特征
│   └── 2.1.3 污水管网概况
├── 2.2 样品采集与处理 (1500-2000字)
│   ├── 2.2.1 采样点布设
│   ├── 2.2.2 采样时间与频次
│   └── 2.2.3 样品预处理方法
├── 2.3 分析测试方法 (1500-2000字)
│   ├── 2.3.1 固相指标测定
│   ├── 2.3.2 液相指标测定
│   └── 2.3.3 气相指标测定
├── 2.4 数据处理与统计分析 (1000-1500字)
│   ├── 2.4.1 数据预处理
│   ├── 2.4.2 统计分析方法
│   └── 2.4.3 软件与工具
└── 2.5 质量控制与保证 (500字)
```

**可复用模板（采样方法）：**
```
本研究于[时间]在[地点]开展采样工作。根据[布点依据]，
在[区域]设置了[数量]个采样点，分别为[S1-Sn]。
采样点具体位置如图[X]所示。采样过程中，
[气体/液体/固体]样品分别采用[具体方法/设备]采集，
采集后立即[保存/处理方法]，并在[时间]内完成[分析/测定]。
每个采样点设置[n]个平行样，以确保数据的可靠性。
```

**常见错误：**
- 方法描述不完整，无法复现
- 缺少质量控制措施说明
- 统计方法选择依据未交代
- 设备型号和参数信息不全

#### 第3章 结果与分析 (约8000-12000字)
```
第3章 结果与分析
├── 3.1 固相碳污染物赋存特征 (2000-3000字)
│   ├── 3.1.1 空间分布特征
│   └── 3.1.2 季节变化特征
├── 3.2 液相碳污染物赋存特征 (2000-3000字)
│   ├── 3.2.1 空间分布特征
│   └── 3.2.2 季节变化特征
├── 3.3 气相碳污染物赋存特征 (2000-3000字)
│   ├── 3.3.1 空间分布特征
│   └── 3.3.2 季节变化特征
├── 3.4 多相态碳污染物关联分析 (1500-2000字)
│   ├── 3.4.1 相关性分析
│   └── 3.4.2 主成分分析
└── 3.5 碳平衡分析 (1500-2000字)
```

**写作模式：** 先描述现象，再解释原因

**可复用模板（描述数据）：**
```
由图[X]可知，[指标]在[区域/时间]呈现出[变化趋势]的特征。
具体而言，[指标]的变化范围为[X-Y]，平均值为[Z±SD]。
其中，[最高值点]的[指标]最高，达到[X]；
[最低值点]的[指标]最低，仅为[Y]。
方差分析结果表明，[不同组别]之间[指标]存在显著差异
(F=[值], p<[0.05/0.01])。
```

**可复用模板（空间对比）：**
```
从空间分布来看，[指标]呈现出明显的[空间规律]。
[区域A]的[指标]显著高于/低于[区域B]
([具体值] vs [具体值], p<[值])，
这可能与[原因1]和[原因2]有关。
[区域A]主要为[功能描述]，[污染物来源/影响因素]较为集中；
而[区域B]则以[功能描述]为主，[影响因素]相对较少。
```

**常见错误：**
- 只描述数据不解释原因
- 图表与文字描述不一致
- 过度解读统计显著性
- 缺少必要的统计检验结果

---

#### 第4章 讨论 (约5000-8000字)
```
第4章 讨论
├── 4.1 [核心发现1的讨论] (1500-2000字)
├── 4.2 [核心发现2的讨论] (1500-2000字)
├── 4.3 [核心发现3的讨论] (1500-2000字)
├── 4.4 研究局限性 (500-800字)
└── 4.5 研究展望 (500-800字)
```

**写作模式：** 讨论 = 本研究发现 + 与文献对比 + 机制解释

**可复用模板（发现+对比）：**
```
本研究发现，[核心发现描述]。这一结果与[文献1]的研究结果
[一致/不一致]。[文献1]在[研究对象]中也观察到了类似现象，
并将其归因于[原因]。然而，[文献2]在[不同条件下]的研究
得出了不同的结论，[具体差异]。这种差异可能源于[原因分析]，
包括[因素1]、[因素2]等。
```

**可复用模板（机制解释）：**
```
[发现]的可能机制如下：首先，[机制1]。[详细解释]。
其次，[机制2]。[详细解释]。
此外，[机制3]也可能起到一定作用。[详细解释]。
上述机制的综合作用可能导致了本研究观察到的[发现]。
```

**常见错误：**
- 讨论只是重复Results的内容
- 与文献对比时选择性引用
- 机制解释缺乏依据或过度推断
- 忽略与预期不符的结果

---

#### 第5章 结论与展望 (约1000-2000字)
```
第5章 结论与展望
├── 5.1 主要结论 (800-1200字)
└── 5.2 不足与展望 (500-800字)
```

**可复用模板（结论）：**
```
本研究以[研究对象]为研究区域，采用[方法]，
对[内容]进行了系统研究。主要结论如下：

(1) [结论1：核心发现1]。[1-2句展开说明]。

(2) [结论2：核心发现2]。[1-2句展开说明]。

(3) [结论3：核心发现3]。[1-2句展开说明]。

上述研究结果为[应用/理论意义]提供了科学依据。
```

**可复用模板（展望）：**
```
本研究仍存在以下不足：
(1) [局限性1]，今后可通过[改进方案]进一步完善；
(2) [局限性2]，后续研究可[扩展方向]。
未来研究可从以下方面深入：
(1) [展望方向1]；
(2) [展望方向2]。
```

## SCI论文结构模板

### 总体结构（IMRaD）

```
Title (标题)
Abstract (摘要, 200-300 words)
  ├── Background
  ├── Methods
  ├── Results
  └── Conclusions
Introduction (引言, 500-800 words)
Materials and Methods (材料与方法, 800-1200 words)
Results (结果, 1500-2500 words)
Discussion (讨论, 1500-2500 words)
Conclusions (结论, 150-200 words)
Acknowledgements
References
Supplementary Materials (可选)
```

### SCI各部分写作要点

#### Title（标题）
**规则：**
- 10-15个实词为宜
- 避免缩写（除非广为人知）
- 包含关键变量和研究范围
- 避免问句式标题

**模板：**
```
[方法/手段] of [研究对象] in [研究范围]:
[关键发现/特征]
```

**本课题示例：**
```
Occurrence and Distribution of Carbon Pollutants
in Solid-Liquid-Gas Phases in Campus Sewage Networks:
A Carbon Balance Analysis
```

#### Abstract（摘要）
**结构：1+1+2+1+1**
```
[1句] Background: [研究背景/问题]
[1句] Objectives: [研究目的]
[2句] Methods: [研究方法和数据来源]
[2-3句] Results: [主要发现，含关键数据]
[1句] Conclusions: [核心结论和意义]
```

**时态：**
- Background/Conclusions: 现在时
- Methods/Results: 过去时

#### Introduction（引言）
**结构：漏斗模型**
```
Paragraph 1: 大背景（宏观→具体领域）
Paragraph 2: 文献综述（现有研究做了什么）
Paragraph 3: 研究空白（还缺什么）
Paragraph 4: 本研究（目的+方法+创新点+结构安排）
```

**高级写法：** 用"However, ... remains unclear/largely unknown"引出研究空白

#### Materials and Methods
**时态：** 全部过去时
**语态：** 被动语态为主

#### Results
**时态：** 过去时
**原则：** 客观描述数据，不做推断（推断留给Discussion）

#### Discussion
**结构：漏斗模型（与Introduction反向）**
```
Paragraph 1: 核心发现概述
Paragraph 2-N: 逐个讨论主要发现（对比文献+机制解释）
Last-1 paragraph: 局限性
Last paragraph: 展望/结论
```

## 硕士论文各章字数分配参考

| 章节 | 字数范围 | 占比建议 |
|------|---------|---------|
| 摘要 | 500-800字 | - |
| 第1章 绪论 | 5000-8000 | 10-15% |
| 第2章 材料与方法 | 5000-8000 | 10-15% |
| 第3章 结果与分析 | 10000-15000 | 25-30% |
| 第4章 讨论 | 5000-8000 | 15-20% |
| 第5章 结论与展望 | 1500-2500 | 5% |
| 参考文献 | 80-120篇 | - |
| 总字数 | 30000-50000 | 100% |

## 技术路线图模板

```
研究区域概况调研
    ↓
采样点布设 → 固相样品 → [测定方法1]
    ↓         液相样品 → [测定方法2]
多季节采样   气相样品 → [测定方法3]
    ↓
数据整理与质控
    ↓
┌──────────┬──────────┬──────────┐
↓          ↓          ↓          ↓
描述统计   空间分析   多变量分析  碳平衡
(均值±SD)  (分布图)   (PCA/HCA)  (质量守恒)
    ↓          ↓          ↓          ↓
└──────────┴──────────┴──────────┘
    ↓
综合讨论与机理解释
    ↓
结论与展望
```
