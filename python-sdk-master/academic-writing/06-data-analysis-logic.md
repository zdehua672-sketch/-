# 数据分析逻辑库

## 数据分析的完整叙事链

```
原始数据 → 数据预处理 → 描述统计 → 正态性检验
  → 差异性检验 → 相关性分析 → 降维分析(PCA) → 聚类分析(HCA)
  → 回归分析 → 综合解读 → 论文文字
```

每一步都有对应的"数据→文字"转换逻辑。

---

## 一、描述性统计 → 文字

### 分析目的
展示数据的基本分布特征（集中趋势 + 离散程度）

### 关键指标
- 均值（Mean）：数据的中心位置
- 标准差（SD）：数据的离散程度
- 最小值/最大值：数据范围
- 中位数（Median）：偏态数据的中心
- 变异系数（CV）：相对离散程度，CV > 100% 为高变异

### 数据→文字模板

**基础描述：**
```
[指标]的统计结果如表[X]所示。
[指标]的变化范围为[最小值]~[最大值]，
平均值为[均值]±[标准差](n=[样本数])。
其中，[最高点]的[指标]最高([值])，
[最低点]的[指标]最低([值])。
```

**含中位数（偏态数据）：**
```
[指标]呈偏态分布（偏度=[值]，峰度=[值]），
中位数为[值](IQR: [下四分位数]-[上四分位数])，
与均值([值])存在一定偏差，表明数据分布[右偏/左偏]。
```

**含变异系数：**
```
[指标]的变异系数为[值]%，属于[低/中/高]变异
（CV < 15%为低变异，15-36%为中等变异，> 36%为高变异），
表明[指标]在不同采样点之间[变化较小/变化较大]。
```

### 本课题示例
```
校园污水管网各采样点液相碳指标的统计结果如表3-1所示。
TOC的变化范围为12.3~156.8 mg/L，平均值为68.5±42.3 mg/L(n=36)。
其中，生活区食堂排水口的TOC最高(156.8 mg/L)，
教学实验区雨水井的TOC最低(12.3 mg/L)。
TOC的变异系数为61.8%，属于高变异，
表明不同功能区和不同采样点之间有机碳含量差异较大。
```

## 二、正态性检验 → 文字

### 分析目的
判断数据是否服从正态分布，决定后续使用参数检验还是非参数检验

### 方法选择
- Shapiro-Wilk检验：样本量 < 50
- K-S检验（Kolmogorov-Smirnov）：样本量 >= 50
- 偏度/峰度辅助判断：|偏度| < 2 且 |峰度| < 7 通常可接受

### 判断标准
- p > 0.05：不拒绝正态假设 → 使用参数检验（t检验/ANOVA）
- p <= 0.05：拒绝正态假设 → 使用非参数检验（Mann-Whitney U/Kruskal-Wallis）

### 数据→文字模板

**正态分布：**
```
Shapiro-Wilk检验结果表明，[指标]在各组中均服从正态分布
(p > 0.05)(表[X])，满足参数检验的前提条件。
因此，后续组间差异分析采用[参数检验方法]。
```

**非正态分布：**
```
Shapiro-Wilk检验结果显示，[指标]在[某些组别]中不服从正态分布
(p = [值] < 0.05)。因此，组间差异分析采用非参数检验方法
([Mann-Whitney U/Kruskal-Wallis检验])。
```

**混合情况：**
```
正态性检验结果表明(表[X])，[指标A]和[指标B]服从正态分布(p > 0.05)，
而[指标C]和[指标D]不服从正态分布(p < 0.05)。
因此，对[指标A]和[指标B]采用参数检验，
对[指标C]和[指标D]采用非参数检验。
```

## 三、差异性检验 → 文字

### 两组比较

**t检验（独立样本，正态分布）：**
```
独立样本t检验结果表明，[组A]与[组B]的[指标]
存在显著差异(t=[自由度]=值, p=值)。
[组A]的[指标]([均值]±[SD])显著高于/低于
[组B]([均值]±[SD])。
```

**Mann-Whitney U检验（非正态分布）：**
```
Mann-Whitney U检验结果显示，[组A]与[组B]的[指标]
差异具有统计学意义(U=值, Z=值, p=值)。
[组A]的[指标]中位数([值])显著高于/低于
[组B]的中位数([值])。
```

### 多组比较

**单因素ANOVA（正态分布）：**
```
单因素方差分析结果表明，不同[分组因素]之间[指标]
存在显著差异(F([df1],[df2])=值, p=值)。
事后多重比较(Tukey HSD)显示，[组A]显著高于[组B](p=值)
和[组C](p=值)，而[组B]与[组C]之间差异不显著(p=值)。
```

**Kruskal-Wallis检验（非正态分布）：**
```
Kruskal-Wallis检验结果表明，不同[分组因素]之间[指标]
存在显著差异(H=值, p=值)。
事后成对比较(Dunn检验，Bonferroni校正)显示，
[组A]显著高于[组B](校正p=值)。
```

### 本课题示例
```
独立样本t检验结果表明，冬季与春季的液相TOC存在显著差异
(t₇₀=3.24, p=0.002)。冬季TOC(85.6±38.2 mg/L)显著高于
春季TOC(52.3±29.7 mg/L)。这一差异可能与冬季低温条件下
微生物分解速率降低以及春季降雨稀释效应有关。
```

### 注意事项
- 报告自由度（df）
- 报告检验统计量（t值/F值/U值/H值）
- 报告精确p值（或p < 0.05 / p < 0.01）
- 如果做了多重比较，需要说明校正方法
- 显著性标注：* p<0.05, ** p<0.01, *** p<0.001

## 四、相关性分析 → 文字

### Pearson相关（线性关系，正态数据）

**数据→文字模板：**
```
Pearson相关性分析结果表明(图[X])，
[变量A]与[变量B]之间呈显著正/负相关(r=值, p<值)。
这表明[变量A]越高/低，[变量B]也趋于越高/低。
从生物学/化学角度，这可能反映了[解释]。
```

### Spearman相关（单调关系，非正态数据）

**数据→文字模板：**
```
Spearman秩相关分析显示，[变量A]与[变量B]之间
存在显著的正/负相关关系(ρ=值, p<值)。
这一结果表明，两者之间存在[线性/单调]的关联趋势，
可能与[解释]有关。
```

### 相关系数的解读标准

| |r| 范围 | 相关强度 | 用于论文的表述 |
|-----------|---------|--------------|
| 0.00-0.29 | 弱相关 | 微弱的正/负相关 |
| 0.30-0.59 | 中等相关 | 中等程度的正/负相关 |
| 0.60-0.79 | 强相关 | 较强的正/负相关 |
| 0.80-1.00 | 极强相关 | 极强的正/负相关 |

### 相关性矩阵的整体描述

```
相关性矩阵分析结果如图[X]所示。
在所有变量对中，[n]对呈显著正相关，[m]对呈显著负相关。
其中，相关性最强的是[变量A]与[变量B](r=值)，
最弱的是[变量C]与[变量D](r=值)。
值得注意的是，[某些变量]之间相关性不显著，
表明它们的变化可能受不同因素控制。
```

### 本课题示例
```
Pearson相关分析结果表明(图3-8)，
液相TOC与DOC之间呈极强正相关(r=0.92, p<0.001)，
这与TOC主要由DOC贡献的认识一致。
固相SOC与液相TOC之间呈中等正相关(r=0.56, p=0.003)，
表明固-液两相碳之间可能存在迁移转化关系。
而气相CH₄与固相SOC之间的相关性不显著(r=0.18, p=0.23)，
提示气态碳排放的控制因素可能较为复杂。
```

## 五、PCA分析 → 文字

### PCA完整叙事逻辑

```
第1层：总体结果（提取了几个主成分，累计方差）
  ↓
第2层：各主成分的变量载荷（哪些变量贡献大）
  ↓
第3层：主成分的物理/化学含义（解读）
  ↓
第4层：样本在主成分空间的分布（得分图解读）
  ↓
第5层：综合意义（揭示了什么规律）
```

### 数据→文字模板

**总体结果：**
```
主成分分析(PCA)结果表明(图[X])，
前[n]个主成分的方差贡献率分别为[X]%, [Y]%, [Z]%
(表[X])，累计方差贡献率达[值]%，基本涵盖了原始变量的主要信息。
```

**变量载荷解读：**
```
第一主成分(PC1)的方差贡献率为[X]%，
主要由[变量1](载荷=值)、[变量2](载荷=值)和
[变量3](载荷=值)贡献。
这些变量均与[过程/来源]密切相关，
因此PC1可以解释为[解释]。
第二主成分(PC2)的方差贡献率为[Y]%，
主要由[变量4](载荷=值)和[变量5](载荷=值)贡献，
反映了[解释]。
```

**得分图解读：**
```
从PCA得分图(图[X])可以看出，[组A]和[组B]的样本
在PC1方向上明显分离，表明两者在[方面]存在本质差异。
[区域A]的样本主要分布在PC1的[正/负]方向，
[区域B]的样本主要分布在PC1的[正/负]方向，
进一步印证了[发现]。
个别样本点（如[样本名]）偏离主群体较远，
可能受到[特殊因素]的影响。
```

**综合意义：**
```
PCA结果表明，校园污水管网碳污染物的赋存特征主要受
[因素1]（PC1）和[因素2]（PC2）两个方面控制。
其中，[因素1]是主导因素，解释了[X]%的总变异。
```

### 本课题示例
```
主成分分析(PCA)结果表明(图3-9)，
前2个主成分的方差贡献率分别为42.3%和28.7%，
累计方差贡献率达71.0%，基本涵盖了原始碳指标的主要信息。

PC1(42.3%)主要由TOC(载荷0.89)、DOC(载荷0.85)和
SOC(载荷0.78)贡献，这些指标均与有机碳含量密切相关，
因此PC1可解释为"有机碳因子"，反映了有机碳在各相态中的
整体丰度。

PC2(28.7%)主要由CH₄(载荷0.82)和CO₂(载荷0.76)贡献，
而与固相和液相指标的相关性较弱，因此PC2可解释为
"气态碳排放因子"，反映了微生物厌氧代谢产气的能力。

从PCA得分图可以看出，生活区样本在PC1方向上明显偏正，
而教学实验区样本偏负，表明生活区有机碳输入更为丰富。
冬季样本在PC2方向上偏正，可能与冬季厌氧条件增强有关。
```

## 六、HCA聚类分析 → 文字

### HCA叙事逻辑

```
第1层：整体聚类结果（分成了几类）
  ↓
第2层：各类别的组成（哪些样本/变量聚在一起）
  ↓
第3层：聚类的物理解释（为什么这些样本/变量聚在一起）
  ↓
第4层：与PCA结果的交叉验证
```

### 数据→文字模板

**样本聚类：**
```
层次聚类分析(HCA)结果如图[X]所示。
在[距离阈值/距离准则]下，[n]个样本可分为[k]类：
第I类包括[样本列表]，主要来自[区域/条件]；
第II类包括[样本列表]，主要来自[区域/条件]；
第III类包括[样本列表]，主要来自[区域/条件]。
聚类结果与PCA得分图的分组趋势基本一致，
相互验证了[分组规律]。
```

**变量聚类：**
```
变量层次聚类结果表明(图[X])，
[变量A]和[变量B]首先聚为一类，
表明两者具有较高的相似性/相关性，可能受相同因素控制。
[变量C]和[变量D]聚为另一类，反映了[解释]。
[变量E]与其他变量距离较远，表明其变化模式较为独立。
```

### 本课题示例
```
层次聚类分析(HCA)结果如图3-10所示。
在Ward距离准则下，36个样本可分为3类：
第I类包括生活区的12个样本（6个冬季+6个春季），
其碳污染物含量整体较高；
第II类包括教学实验区的12个样本，碳含量居中；
第III类包括办公区的12个样本，碳含量整体较低。
聚类结果与PCA得分图的分组趋势高度一致，
进一步证实了功能区类型是影响碳赋存特征的主导因素。
值得注意的是，冬季和春季样本并未各自聚为一类，
而是按功能区分布，表明空间差异大于季节差异。
```

## 七、回归分析 → 文字

### 一元线性回归

**数据→文字模板：**
```
一元线性回归分析结果表明，
[自变量]对[因变量]具有显著的预测作用
(R²=值, F(1,[df])=值, p=值)。
回归方程为：[因变量] = [截距] + [斜率] × [自变量]。
[自变量]每增加1个单位，[因变量]平均增加/减少[斜率]个单位。
```

### 多元线性回归

**数据→文字模板：**
```
多元线性回归分析结果如表[X]所示。
模型整体显著(R²=值, 调整R²=值, F([df1],[df2])=值, p<值)，
说明纳入的自变量可以解释[因变量][X]%的变异。
各变量的标准化回归系数(β)显示：
[变量1]的影响最为显著(β=值, p<值)，
其次是[变量2](β=值, p<值)，
而[变量3]的影响不显著(β=值, p=值)。
这表明[因变量]的变化主要受[主要因素]驱动。
```

### VIF共线性诊断

```
回归模型的多重共线性诊断表明，各变量的方差膨胀因子(VIF)
均小于[10/5]（表[X]），表明模型不存在严重的多重共线性问题。
```

## 八、污染物来源解析 → 文字

### PCA/PMF来源解析叙事

```
来源解析的叙事框架：
1. 识别出几个来源
2. 每个来源的特征标志物
3. 每个来源的贡献比例
4. 来源的空间/时间差异
5. 与已有研究的对比
```

**模板：**
```
基于PCA分析，本研究识别出[n]个主要碳污染物来源：
来源1为[来源名称]，其特征标志物为[变量]，
贡献率为[X]%；
来源2为[来源名称]，特征标志物为[变量]，
贡献率为[Y]%。
从空间分布来看，[来源1]在[区域]贡献较大，
而[来源2]在[区域]更为突出。
这一结果与[文献]在[类似环境]中的来源解析结论一致/有所差异。
```

### 本课题示例
```
基于PCA分析和相关性矩阵，本研究识别出校园污水管网碳污染物
的3个主要来源：
(1) 生活有机碳来源：以TOC、DOC、BOD₅为特征指标，
    主要来源于食堂排水、宿舍生活污水，贡献率约42%；
(2) 微生物代谢来源：以CH₄、CO₂为特征指标，
    主要来源于沉积物厌氧分解和管壁生物膜代谢，贡献率约29%；
(3) 地表径流来源：以POC、SS为特征指标，
    主要来源于降雨冲刷地表有机碎屑，贡献率约15%。
```

## 九、时空分布分析 → 文字

### 空间分布叙事

```
空间分布叙事框架：
1. 总体趋势（高→低 / 上游→下游）
2. 热点区域识别
3. 空间差异的统计检验
4. 空间差异的原因解释
5. 与土地利用/功能区的关系
```

**模板：**
```
[指标]的空间分布呈现明显的[规律]特征（图[X]）。
从[方向1]到[方向2]，[指标]呈递增/递减趋势。
[热点区域]的[指标]最高，达到[值]，
显著高于其他区域(p<值)。
这一分布格局与[原因]密切相关。
[热点区域]为[功能]，[污染物来源/影响因素]集中，
而[其他区域]则[相反情况]。
```

### 时间变化叙事

```
时间变化叙事框架：
1. 总体趋势（季节对比）
2. 峰值/谷值出现的时间
3. 时间变化的统计检验
4. 时间变化的原因解释（温度/降雨/活动模式）
```

**模板：**
```
从时间变化来看，[指标]呈现明显的季节性差异（图[X]）。
冬季[指标]([均值]±[SD])显著高于/低于春季
([均值]±[SD])(t=值, p=值)。
这一季节差异主要受以下因素驱动：
(1) 温度效应：冬季低温导致[过程]减缓/增强；
(2) 稀释效应：春季降雨增加了[稀释/冲刷]作用；
(3) 活动模式：[具体说明]。
```

```
碳平衡叙事框架：

```

**模板：**
```
碳平衡分析结果如图[X]和表[X]所示。
在本研究的系统边界内，碳输入主要通过[途径1]([通量值])和
[途径2]([通量值])实现，碳输出主要通过[途径3]([通量值])和

主要来源于[不确定性来源]。
```

### 本课题示例

## 十一、统计结果→文字的通用规则

### 报告格式规范

| 统计内容 | 报告格式 | 示例 |
|---------|---------|------|
| 均值±标准差 | X±SD | 68.5±42.3 mg/L |
| 中位数(四分位距) | Median (IQR) | 58.2 (35.1-89.6) mg/L |
| t检验 | t(df)=值, p=值 | t(70)=3.24, p=0.002 |
| ANOVA | F(df1,df2)=值, p=值 | F(2,33)=8.56, p=0.001 |
| Mann-Whitney U | U=值, Z=值, p=值 | U=245, Z=-2.31, p=0.021 |
| Pearson相关 | r(df)=值, p=值 | r(34)=0.92, p<0.001 |
| Spearman相关 | ρ=值, p=值 | ρ=0.78, p<0.001 |
| 回归 | β=值, p=值 | β=0.65, p=0.003 |
| 卡方检验 | χ²(df)=值, p=值 | χ²(2)=6.34, p=0.042 |

### 精度规则
- 浓度值：保留1-2位小数（如68.5 mg/L）
- 百分比：保留1位小数（如42.3%）
- p值：< 0.001 时写 p < 0.001；否则报告精确值（p=0.023）
- 相关系数：保留2-3位小数（r=0.92）
- R²：保留2-3位小数（R²=0.847）

### 显著性标注规范
```
* p < 0.05
** p < 0.01
*** p < 0.001
ns 不显著 (not significant)
```

## 十二、综合分析叙事模板

### 从单一分析到综合解读

```
单独的统计结果是碎片化的。综合叙事需要：
1. 描述性统计 → 总体特征是什么
2. 差异性检验 → 哪些差异是显著的
3. 相关性分析 → 变量之间有什么关系
4. PCA → 变量可以归纳为几个因子
5. HCA → 样本/变量如何分组
6. 综合解读 → 以上所有结果共同说明了什么

综合叙事模板：
综合以上分析结果，可以得出以下认识：
(1) [从描述统计得出的认识]；
(2) [从差异检验得出的认识]；
(3) [从相关分析和PCA得出的认识]；
(4) [从HCA得出的认识]。
上述认识共同指向[核心结论]。
```
