# 06-data-analysis-logic


## ### 方法选择

- Shapiro-Wilk检验：样本量 < 50
- K-S检验（Kolmogorov-Smirnov）：样本量 >= 50
- 偏度/峰度辅助判断：|偏度| < 2 且 |峰度| < 7 通常可接受

### 判断标准
- p > 0.05：不拒绝正态假设 → 使用参数检验（t检验/ANOVA）
- p <= 0.05：拒绝正态假设 → 使用非参数检验（Mann-Whitney U/Kruskal-Wallis）

### 数据→文字模板

**正态分布：**
```
Shapiro-Wilk检验结果表明，[指标]在各组中均服从正态分布
(p > 0.05)(表[X])，满足参数检验的前提条件。
因此，后续组间差异分析采用[参数检验方法]。
```

**非正态分布：**
```
Shapiro-Wilk检验结果显示，[指标]在[某些组别]中不服从正态分布
(p = [值] < 0.05)。因此，组间差异分析采用非参数检验方法
([Mann-Whitney U/Kruskal-Wallis检验])。
```

**混合情况：**
```
正态性检验结果表明(表[X])，[指标A]和[指标B]服从正态分布(p > 0.05)，
而[指标C]和[指标D]不服从正态分布(p < 0.05)。
因此，对[指标A]和[指标B]采用参数检验，
对[指标C]和[指标D]采用非参数检验。
```

## 三、差异性检验 → 文字

### 两组比较

**t检验（独立样本，正态分布）：**
```
独立样本t检验结果表明，[组A]与[组B]的[指标]
存在显著差异(t=[自由度]=值, p=值)。
[组A]的[指标]([均值]±[SD])显著高于/低于
[组B]([均值]±[SD])。
```

**Mann-Whitney U检验（非正态分布）：**
```
Mann-Whitney U检验结果显示，[组A]与[组B]的[指标]
差异具有统计学意义(U=值, Z=值, p=值)。
[组A]的[指标]中位数([值])显著高于/低于
[组B]的中位数([值])。
```

### 多组比较

**单因素ANOVA（正态分布）：**
```
单因素方差分析结果表明，不同[分组因素]之间[指标]
存在显著差异(F([df1],[df2])=值, p=值)。
事后多重比较(Tukey HSD)显示，[组A]显著高于[组B](p=值)
和[组C](p=值)，而[组B]与[组C]之间差异不显著(p=值)。
```

**Kruskal-Wallis检验（非正态分布）：**
```
Kruskal-Wallis检验结果表明，不同[分组因素]之间[指标]
存在显著差异(H=值, p=值)。
事后成对比较(Dunn检验，Bonferroni校正)显示，
[组A]显著高于[组B](校正p=值)。
```

### 本课题示例
```
独立样本t检验结果表明，冬季与春季的液相TOC存在显著差异
(t₇₀=3.24, p=0.002)。冬季TOC(85.6±38.2 mg/L)显著高于
春季TOC(52.3±29.7 mg/L)。这一差异可能与冬季低温条件下
微生物分解速率降低以及春季降雨稀释效应有关。
```

### 注意事项
- 报告自由度（df）
- 报告检验统计量（t值/F值/U值/H值）
- 报告精确p值（或p < 0.05 / p < 0.01）
- 如果做了多重比较，需要说明校正方法
- 显著性标注：* p<0.05, ** p<0.01, *** p<0.001

## 四、相关性分析 → 文字

### Pearson相关（线性关系，正态数据）

**数据→文字模板：**
```
Pearson相关性分析结果表明(图[X])，
[变量A]与[变量B]之间呈显著正/负相关(r=值, p<值)。
这表明[变量A]越高/低，[变量B]也趋于越高/低。
从生物学/化学角度，这可能反映了[解释]。
```

### Spearman相关（单调关系，非正态数据）

**数据→文字模板：**
```
Spearman秩相关分析显示，[变量A]与[变量B]之间
存在显著的正/负相关关系(ρ=值, p<值)。
这一结果表明，两者之间存在[线性/单调]的关联趋势，
可能与[解释]有关。
```

### 相关系数的解读标准

| |r| 范围 | 相关强度 | 用于论文的表述 |
|-----------|---------|--------------|
| 0.00-0.29 | 弱相关 | 微弱的正/负相关 |
| 0.30-0.59 | 中等相关 | 中等程度的正/负相关 |
| 0.60-0.79 | 强相关 | 较强的正/负相关 |
| 0.80-1.00 | 极强相关 | 极强的正/负相关 |

### 相关性矩阵的整体描述

```
相关性矩阵分析结果如图[X]所示。
在所有变量对中，[n]对呈显著正相关，[m]对呈显著负相关。
其中，相关性最强的是[变量A]与[变量B](r=值)，
最弱的是[变量C]与[变量D](r=值)。
值得注意的是，[某些变量]之间相关性不显著，
表明它们的变化可能受不同因素控制。
```

### 本课题示例
```
Pearson相关分析结果表明(图3-8)，
液相TOC与DOC之间呈极强正相关(r=0.92, p<0.001)，
这与TOC主要由DOC贡献的认识一致。
固相SOC与液相TOC之间呈中等正相关(r=0.56, p=0.003)，
表明固-液两相碳之间可能存在迁移转化关系。
而气相CH₄与固相SOC之间的相关性不显著(r=0.18, p=0.23)，
提示气态碳排放的控制因素可能较为复杂。
```

```
