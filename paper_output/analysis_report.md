## 分析决策

- ✓ **descriptive**: 基础分析，所有论文必备
- ✓ **normality**: 决定参数/非参数检验选择
- ✓ **group_comparison**: 检测到2个季节组(冬季,春季)，需检验差异
- ✓ **correlation**: 25个变量，适合做相关性分析
- ✓ **pca**: 40个样本×25个变量，适合降维
- ✓ **hca**: 40个样本，适合聚类分析
- ✓ **regression**: 发现3对有科研意义的变量组合
- ✓ **carbon_balance**: 检测到2相碳数据，可做碳平衡分析

### 正态性检验结果

Shapiro-Wilk正态性检验结果表明，11个变量符合正态分布(p>0.05)，14个变量不符合正态分布(p≤0.05)。
非正态变量包括：CH$_4$ (ppm)、N$_2$O (ppm)、CO$_2$ (ppm)、CO2本底值、O$_2$ (%vol)等。
对于非正态分布的变量，后续组间比较采用Mann-Whitney U非参数检验。
