# 01-sci-rules


## ### Abstract（摘要）

**结构：Background(1) + Objective(1) + Methods(1-2) + Results(2-3) + Conclusions(1)**
**字数：** 200-300 words（期刊要求不同）
**时态：** Background/Conclusions 现在时；Methods/Results 过去时

**模板：**
```
[Background] Carbon pollutant distribution in multi-phase sewage systems
is critical for accurate carbon accounting, yet remains poorly understood
in campus sewage networks.

[Objective] This study investigated the occurrence and distribution of
carbon pollutants in solid, liquid, and gas phases in a campus sewage
network, and conducted a carbon balance analysis.

[Methods] Samples were collected from [n] sampling sites during winter
and spring. Solid-phase, liquid-phase, and gas-phase carbon indicators
were determined using [methods]. Multivariate statistical analyses
including PCA and HCA were employed.

[Results] TOC concentrations ranged from [X] to [Y] mg/L, with
significantly higher values in winter than in spring (p < 0.05).
PCA identified [n] principal components explaining [X]% of the total
variance. The carbon balance revealed that [finding].

[Conclusions] This study provides new insights into multi-phase carbon
dynamics in campus sewage networks, with implications for urban carbon
management.
```

**常见错误：**
- Abstract中出现图表引用（"as shown in Figure 1"）
- 方法描述过于详细
- 结果缺乏具体数据
- 使用缩写未定义

## ### Introduction（引言）

**结构：漏斗模型（4段式）**

**Paragraph 1 - 大背景（3-5句）：**
```
[大背景句] [领域重要性] [具体问题的引出]
```
- 从全球/国家层面切入
- 逐步聚焦到具体领域
- 引出核心问题

**Paragraph 2 - 文献综述（5-8句）：**
```
[前人做了什么] [主要发现] [使用的典型方法]
```
- 按时间/主题组织
- 不是简单罗列，要有批判性
- 突出研究趋势

**Paragraph 3 - 研究空白（3-5句）：**
```
Despite these efforts, [空白1] remains unclear.
Moreover, [空白2] has received limited attention.
In particular, [空白3] needs further investigation.
```
- 用"However/Despite/Although"引出空白
- 空白要具体，不要说"很少有人研究"
- 空白要与本研究直接相关

**Paragraph 4 - 本研究（3-5句）：**
```
Therefore, this study aims to [目标].
Specifically, [具体目标1]; [具体目标2]; [具体目标3].
To achieve these objectives, [简述方法].
The results will provide [预期贡献].
```

**Introduction常用信号词：**
- 引出背景：Recently, In recent decades, With the increasing...
- 引出文献：Several studies have shown that..., Previous research has demonstrated...
- 引出空白：However, ..., remains unclear/limited/poorly understood
- 引出目的：Therefore, the aims of this study were to...

## ### Materials and Methods（材料与方法）

**时态：** 全部过去时
**语态：** 被动语态为主
**原则：** 详细到可重复

**应包含的信息：**
1. 研究区域概况（位置、气候、特征）
2. 采样方案（时间、地点、方法、数量）
3. 分析方法（仪器型号、参数、标准）
4. 数据处理（软件、统计方法）
5. 质量控制（平行样、空白样、回收率）

**模板：**
```
2.1 Study area
[Location], [climate], [sewage system characteristics].

2.2 Sample collection
Samples were collected at [n] sites during [period].
At each site, [replicates] were collected.
[保存方法]. Samples were transported to the laboratory under [条件].

2.3 Analytical methods
[Parameter A] was determined using [instrument] ([manufacturer], [model])
according to [standard]. [Parameter B] was measured by [method].
详细参数设置...

2.4 Statistical analysis
Data were analyzed using [software] ([version]).
[具体统计方法].
A p-value < 0.05 was considered statistically significant.

2.5 Quality assurance and quality control
[QA/QC措施].
Method blanks, duplicate samples, and standard reference materials
were used for quality control. Recovery rates ranged from [X]% to [Y]%.
```

**常见错误：**
- 缺少设备型号和参数
- 未说明质量控制措施
- 统计方法未说明选择依据
- 缺少数据预处理步骤（如异常值剔除）

---

**Key Findings:**
- A p-value < 0.05 was considered statistically significant.

## ### Results（结果）

**时态：** 过去时
**原则：** 客观描述，不做推断（推断留给Discussion）

**写作模式：** 先总体后具体，先文字后图表

**组织方式选择：**
- 按指标组织：TOC → DOC → POC → CH₄ → CO₂
- 按主题组织：空间分布 → 时间变化 → 相关性 → PCA
- 按相态组织：固相 → 液相 → 气相 → 多相态关联

**单个结果描述模板：**
```
[指标] [变化趋势] across [分组] (Fig. [X]).
The values ranged from [min] to [max], with a mean of [mean]±[SD] (Table [X]).
[最高点] showed the highest [value], while [最低点] showed the lowest [value].
[差异检验] revealed significant differences among [groups] ([statistic], p = [value]).
```

**图表引用规则：**
- 先在正文中描述结果，然后说"(Fig. X)"或"(Table X)"
- 不要说"as shown in Fig. X"（这个在Discussion中可以用）
- Results中图表应自明（self-explanatory）

## ### Conclusions（结论）

**时态：** 现在时
**字数：** 150-200 words（或3-5个要点）

**模板：**
```
This study systematically investigated [content] in [subject].
The main findings are as follows:
(1) [Finding 1].
(2) [Finding 2].
(3) [Finding 3].
These results provide [contribution] and have implications for [application].
```

**注意：**
- 不要引入新数据
- 不要重复Abstract
- 每条结论要对应Results中的具体发现

## 五、语言规范

### 禁用词/表达

| 禁用 | 替代 | 原因 |
|------|------|------|
| prove | demonstrate/suggest/indicate | 科学研究不能"证明" |
| very | extremely/remarkably 或直接删掉 | 空洞的程度副词 |
| a lot of | considerable/substantial | 不够正式 |
| get | obtain/achieve | 口语化 |
| big/large | significant/substantial | 不够精确 |
| show | reveal/demonstrate/indicate | 更精确 |
| find out | discover/reveal | 更正式 |
| think | suggest/believe/consider | 更学术 |
| good | favorable/appropriate/positive | 模糊 |
| bad | unfavorable/adverse/negative | 模糊 |

### 推荐学术动词

**描述发现：**
- reveal, demonstrate, indicate, suggest, show
- confirm, support, validate
- observe, detect, identify, determine

**表达差异：**
- differ, vary, range
- exceed, surpass
- approach, approximate

**表达关系：**
- correlate, associate, relate
- contribute, attribute
- derive, originate

**表达变化：**
- increase, decrease, fluctuate
- enhance, reduce, diminish
- peak, decline, stabilize

### 缩写规则
- 首次使用写全称，括号内注明缩写：total organic carbon (TOC)
- 之后统一使用缩写
- 标题和摘要中首次出现也需要定义
- 公认缩写可直接使用：DNA, pH, km, etc.

## 六、各期刊风格差异

### 环境科学领域主要期刊

| 期刊 | 影响因子 | 特点 | 投稿偏好 |
|------|---------|------|---------|
| Environmental Science & Technology | ~11 | 领域顶刊 | 创新性+应用性 |
| Water Research | ~12 | 水领域顶刊 | 系统性+深度 |
| Science of the Total Environment | ~9 | 综合性环境 | 广度+数据量 |
| Environmental Pollution | ~8 | 污染方向 | 新颖性 |
| Chemosphere | ~8 | 化学+环境 | 机理研究 |
| Journal of Environmental Sciences | ~6 | 国际化 | 研究完整性 |

### 不同类型论文的要求

**研究型论文（Research Article）：**
- 标准IMRaD结构
- 3000-6000 words
- 5-8个图/表

**短通讯（Short Communication）：**
- 紧凑结构
- 2000-3000 words
- 2-4个图/表

**综述（Review）：**
- 全面的文献覆盖
- 6000-10000 words
- 需要系统的文献搜索策略

## 七、SCI投稿前检查清单

### 格式检查
- [ ] 符合目标期刊的Author Guidelines
- [ ] 字数在规定范围内
- [ ] 图/表数量符合要求
- [ ] 参考文献格式正确
- [ ] 补充材料完整

### 内容检查
- [ ] Title简洁且包含关键信息
- [ ] Abstract包含背景、方法、结果、结论
- [ ] Introduction逻辑清晰（背景→文献→空白→目的）
- [ ] Methods详细到可重复
- [ ] Results客观描述，无推断
- [ ] Discussion解释了"为什么"，不仅重复"是什么"
- [ ] Conclusions与Results对应

### 语言检查
- [ ] 各部分时态正确
- [ ] 被动语态在Methods中使用
- [ ] 无语法错误
- [ ] 无拼写错误
- [ ] 术语使用一致
- [ ] 缩写首次使用已定义

### 图表检查
- [ ] 分辨率 ≥ 300 dpi
- [ ] 字体大小可读（≥ 8pt）
- [ ] 颜色可区分（考虑色盲读者）
- [ ] 图表标题完整
- [ ] 图表在正文中已引用

### 伦理检查
- [ ] 所有作者同意投稿
- [ ] 无一稿多投
- [ ] 数据真实可靠
- [ ] 致谢已获同意
- [ ] 利益冲突声明

### 投稿材料检查
- [ ] Cover Letter
- [ ] Title Page（含作者信息、通讯作者、ORCID）
- [ ] Main Manuscript（双倍行距/符合格式）
- [ ] Figures（单独文件或嵌入）
- [ ] Tables（单独文件或嵌入）
- [ ] Supplementary Materials
- [ ] Highlights（3-5条，每条85字符以内）
- [ ] Graphical Abstract（如需要）
- [ ] Credit Author Statement
- [ ] Declaration of Interest
