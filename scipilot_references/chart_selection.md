## 决策三轴

每次画图前问自己三个问题：

### 轴 1：变量数量与类型

| 组合 | 变量结构 |
|---|---|
| 1×连续 | 看分布 |
| 1×分类 | 看占比 |
| 1×分类 + 1×连续 | 看组间比较 |
| 2×连续 | 看关系 |
| 1×时间 + 1×连续 | 看趋势 |
| 多个连续 | 看相关 |
| 二维矩阵（n×m 数值） | 看模式 |
| 嵌套分组（A 下分 B，B 下分 C） | 看层次 |

### 轴 2：想表达什么（论证意图）

**这是最容易被忽略的一轴。** 同样的数据，论证目标不同，图就不同。

- **分布** — "这组数据长什么样" → 直方图 / KDE / 箱线
- **比较** — "A 比 B 高 / 低 / 不同" → 箱线 / 柱状（带误差）/ 小提琴
- **关系** — "X 越大 Y 越大" → 散点 + 回归
- **趋势** — "随时间/剂量变化" → 折线 + 误差带
- **构成** — "总和被分成几份" → 堆叠柱状（**不要饼图**）
- **相关** — "哪些变量之间相关" → 相关性热力图 / pairplot
- **差异** — "组间差异是否显著" → 箱线 + 显著性标注
- **不确定性** — "估计有多准" → 误差棒 / 置信带

### 轴 3：数据规模

| 样本量 | 建议 |
|---|---|
| 每组 n ≥ 30 | 箱线、柱状（带误差）、小提琴都可以；分布稳定 |
| 每组 10 ≤ n < 30 | 优先箱线/小提琴；柱状必须叠加 stripplot 显示原始点 |
| 每组 3 ≤ n < 10 | **强烈建议**直接散点/stripplot；箱线慎用（四分位估计不可靠） |
| 每组 n < 3 | **不要画箱线/小提琴**——统计估计无意义。直接展示每个点 |
| 数据总数 > 10⁴ | 散点用 alpha=0.1-0.3 防 overplotting，或改用 hexbin/2D KDE |

## 数据形态 → 推荐图型 速查表

| 数据形态 | 首选 | 备选 | 不该用 |
|---|---|---|---|
| 单个连续看分布 | KDE / 直方图 | 箱线 / 小提琴 | 饼图 / 折线 |
| 单个分类看占比 | 横向柱状（按值排序） | 堆叠柱状（单根） | **饼图**（人眼判断角度差） |
| 1 分类 vs 1 连续，n≥10/组 | 箱线 + stripplot | 小提琴 / 带误差柱 | 仅画均值柱（掩盖分布） |
| 1 分类 vs 1 连续，n<10/组 | stripplot / 蜂巢散点 | dot plot | 箱线（估计不可靠）/ 均值柱 |
| 2 连续 | 散点 + 回归线 | 2D KDE / hexbin（大样本） | 折线（除非有序） |
| 时间 vs 连续 | 折线 + 误差带 | 阶梯图 / 散点 | 柱状 |
| 多连续（3-20 个变量） | 相关性热力图 | pairplot | 平行坐标（除非交互） |
| 多连续（>20 个变量） | 热力图 + 聚类 | PCA / UMAP 散点 | pairplot 满屏 |
| 矩阵数据 | 热力图（感知均匀色图）| 表格 | 3D 表面 |
| 构成 | 堆叠柱状 / treemap | 100% 堆叠柱 | **饼图** / **3D 饼图** |
| 二分类预测性能 | ROC / PR 曲线 | 混淆矩阵热力图 | 仅给准确率柱状 |

## 按数据形态分述

### 1. 单个连续变量看分布

**首选**：KDE（核密度估计）—— 平滑、看趋势；样本量小或数据离散用直方图

**备选**：
- **箱线图**：看四分位 + 中位数 + 异常值
- **小提琴图**：箱线 + KDE 合体
- **rug plot 叠加**：在底部画一条每个点的小竖线，n 不大时直接可见

**何时不该用**：n < 5 时所有"分布"图都不可靠，直接列点。

**论证意图区分**：
- "这组数据偏左 / 偏右" → KDE 最清楚
- "75% 的样本落在哪个区间" → 箱线最直接
- "有几个离群极值" → 箱线 + outlier 标记

### 2. 单个分类变量看占比

**首选**：横向柱状图（条形图），按值排序，类别名写在 y 轴。

**饼图的问题**（多家期刊不接受）：
- 人眼对角度的判断比对长度差至少 3 倍
- 类别 > 5 时配色不够用
- 3D 饼图直接灾难——视角扭曲所有占比

**备选**：treemap（类别 > 10 时空间利用率高）

### 3. 一个分类 + 一个连续（组间比较）

**这是最常见的科研图。决策树**：

```
每组样本量 n ?
├── n < 3          → 直接列点（dot plot）；不要画统计图
├── 3 ≤ n < 10     → stripplot / 蜂巢散点 / dot plot；箱线慎用
├── 10 ≤ n < 30   → 箱线/小提琴 + stripplot 叠加（必须！）
└── n ≥ 30         → 箱线 / 小提琴 / 带误差柱状都可
```

**为什么强调"叠加原始点"**：
- 均值柱状图把 n=5 和 n=500 画得一样高
- 箱线把"双峰分布"和"单峰分布"画得一样
- 叠加原始点 = 把分布真相摆出来，审稿人一眼可信

**论证意图区分**：
- "A 显著高于 B" → 箱线 + 显著性桥（* p<0.05）+ 图注交代检验
- "A 和 B 分布重叠很多" → 小提琴 + stripplot，让重叠肉眼可见
- "A 整体趋势是上升" → 折线连均值（如果分组有顺序）

### 4. 两个连续变量看关系

**首选**：散点图 + 回归拟合线 + 95% CI 带

**额外信号**：
- 第三维分类 → `hue`（颜色）+ `style`（marker 形状）双重编码
- 第三维连续 → 颜色映射（必须加 colorbar）；或 marker 大小
- **样本量 > 1000**：alpha=0.1-0.3 防止 overplotting；或改 hexbin / 2D KDE

**何时不该连线**：
- x 是分类变量
- x 是非连续/非有序数值（如 ID 编号）
- 点之间**没有连续关系**（折线暗示了不存在的趋势）

### 5. 时间序列看趋势

**首选**：折线图 + 误差带（SEM 阴影或 95% CI）

**何时该连线**：x 是真正的时间或序列（剂量、年龄段等有连续意义的轴）

**多条线在一张图**：
- 2-3 条：颜色 + 线型双重编码
- 4-6 条：颜色 + marker 双重编码，且在曲线末端直接标名字（省去图例查找）
- > 6 条：拆图，或者用 small multiples（n×n 网格各画一条）

**坑**：
- 不要把"前测/后测"两个时间点画折线——画 paired-dot 更诚实
- y 轴起点要不要从 0 开始？看变化幅度和是否有意义的 0 基线

### 6. 多变量相关

**首选**：相关性热力图（n 列 × n 列）

**配置**：
- 双向数据（正负有意义）→ 发散色图 `RdBu_r` + `center=0`
- 配 annot 显示 r 值
- 大矩阵（>20 列）+ hierarchical clustering 重排行列

**备选**：散点矩阵（pairplot）—— 4-8 个变量内可读，更多就糊

### 7. 矩阵型数据

经典场景：基因表达矩阵、混淆矩阵、特征图、相似度矩阵。

**色图选择**：
- 单向数据（全正）→ `viridis` / `magma` / `inferno` / `cividis`
- 双向数据 → `RdBu_r` / `PiYG` / `seismic` + `center=0`
- **永远不用** `rainbow` / `jet` / `hsv`——感知不均匀

**注意**：
- 行列足够小时配 `annot=True` 显示数字
- 行列大但矩阵稀疏时用 `imshow`，关掉 grid，加 `colorbar`
- 混淆矩阵 + 行归一化（看精确率/召回率）

### 8. 嵌套/交叉分组（多维分组）

例：3 dose × 4 timepoint × 2 sex × 多次重复测一个 response。

**映射策略**——把维度映射到视觉通道：

| 映射 | 优先级 | 容量 |
|---|---|---|
| x 轴 | 1 | 中（5-10 个分类） |
| 颜色 | 2 | 小（3-5 个） |
| marker / 线型 | 3 | 极小（2-3 个） |
| 子图（小倍图 / facet） | 4 | 中（2-8 个面板） |
| y 轴 | y 是响应变量，不参与分组映射 | — |

**规则**：维度数 ≤ 通道容量乘积，否则**拆图**（见下）。

## 同一批数据，不同论点 → 不同图

### 例 1：药物 A 和 B 对响应时间的影响

数据：30 受试者 × 2 药 × 5 时间点 = 300 个测量。

| 论点 | 推荐图 | 为什么 |
|---|---|---|
| "药物 A 整体快于 B" | 箱线（x=药物，y=时间，合并所有时间点） | 把时间维度压平专注组间 |
| "A 和 B 在 t=3 分歧最大" | 折线（x=time，y=mean，hue=药物 + 误差带） | 突出时间动态 |
| "受试者个体差异极大" | spaghetti plot（每个受试者一条细线）+ 粗均线 | 把变异性可视化 |
| "AB 在 t=3 显著不同" | 配对箱线 + 显著性桥（专做 t=3） | 单点对比 + 统计标注 |

**同一份数据 4 张完全不同的图。** 在 Step 0 必须先问清楚论点。

### 例 2：基因表达谱

数据：1000 基因 × 20 样本表达矩阵 + 样本分两组。

| 论点 | 推荐图 |
|---|---|
| "两组样本可分" | PCA / UMAP 散点（hue=组） |
| "哪些基因差异表达" | volcano plot（log2FC vs -log10 p） |
| "某通路的基因都上调" | 子集 heatmap（行=通路基因，列=样本） |
| "样本间整体相似度" | sample-sample 相关热力图 + 聚类 |

## 何时该拆图

### 拆图判断标准

满足以下**任何一条**就该拆：

1. **维度组合数 > 12**：例如 4×4 = 16 个 panel——一张图能塞得下，但读者解读不动
2. **x 轴标签碰撞**：x 轴分类 > 8 个，标签需要 45°+ 旋转才放得下
3. **图例 > 6 项**：超出短期记忆容量，读者要反复来回看
4. **y 轴尺度跨数个量级 + 不能用 log**：拆成两张分量级画
5. **想说两件事**：如"A 比 B 快" + "A 比 B 准"，是两张图

### 拆图策略

- **按分组维度拆**：原图 hue=sex，拆成 fig 1 (male) + fig 2 (female)
- **按 panel 拆**：原图 2×2 panel，拆成 fig 1 (a,b) + fig 2 (c,d)
- **按论点拆**：原图想说三件事，拆成三个独立的小图
- **主图 + 补充材料**：主图只放核心结论，剩下塞 supplementary

## 图型语义边界

### 折线 vs 散点

| | 折线 | 散点 |
|---|---|---|
| 点之间是否有连续关系 | **必须有** | 不必有 |
| x 是 | 时间 / 剂量 / 序列 | 任意 |
| 适合 | 趋势、动态 | 关系、相关 |
| 误判风险 | **连不该连的点**——暗示不存在的趋势 | overplotting |

### 柱状 vs 箱线

| | 柱状（带误差）| 箱线 |
|---|---|---|
| 看什么 | 均值 ± 误差 | 中位数 + 四分位 + 异常值 |
| 假设 | 单峰、近高斯 | 不假设 |
| 适合 n | ≥30 | ≥10 |
| 双峰分布会怎样 | 看不出，被均值吃掉 | 也看不太出（但比柱状好） |
| **最佳实践** | 叠加 stripplot | 叠加 stripplot |

**核心**：n 小或分布形态未知时，**默认箱线 > 柱状**。

### 热力图 vs 散点矩阵

| | 热力图 | 散点矩阵 |
|---|---|---|
| 信息密度 | 高 | 中 |
| 看什么 | 单一统计量（r、表达值） | 双变量分布全貌 |
| 适合维度 | 5-50+ 列 | 2-8 列 |
| 缺点 | 只看一个统计量 | 维度多了糊 |

经验：> 8 个变量直接用热力图；2-8 个用 pairplot 看分布更丰富。
